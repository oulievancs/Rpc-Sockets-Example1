Παράδοση 1ης Άσκησης

Ο φάκελος περιέχει έναν client υλοποιημένο σε γλώσσα C ο οποίος μπορεί να
συνδεθεί με έναν C socket stream Server καθώς και να επικοινωνίσει. Έναν
C socket concurrent stream Server υλοποιημένο σε γλώσσα C, με χρήση των
threads της γλώσσας καθώς και υλοποίηση RPC Server, ο οποίος υλοποιεί 3 συναρτήσεις:
	1) Μέση τιμή διανύσματος
	2) Μέγιστη και ελάχιστη τιμή διανύσματος
	3) Γινόμενο διανύσματος με πραγματικό αριθμό.
	
Η ροή έχει ως εξής, αρχικά ο client-socket αποστέλει το διάνυσμα και
την επιλογή που θέλει να εκτελέσει με βάση τις προηγούμενες λειτουργίες
στον socket server, έπειτα ο server-socket λαμβάνει τα μυνήματα του client
και καλεί μία απο τις παραπάνω λειτουργίες με την βοήθεια του rpc,
λειτουργίες (συναρτήσεις) απομακρυσμένες, στέλνει αίτημα στον αντίστοιχο
rpc server. Ο server socket παίζει ταυτόχρονα και τον ρόλο του
rpc client.

Αρχικά για να λειτουργίσει αυτό το σύστημα πρέπει να κάνουμε compile όλα τα
αρχεία που εξάγαμε απο το rpcgen και τους κώδικες C με την εντολή "make",
υπάρχει κατάλληλο Makefile στον φάκελο. Έπειτα εκτελούμε "make rpc_server"
για να ξεκινήσουμε τον rpc server. Στη συνέχεια τρέχουμε
"make rpc_cl_socket_srv" για να ξεκινήσουμε τον socket server και τέλος
"make socket_client" για να ξεκινήσουμε τον socket client. Το μενού και
όλα τα αποτελέσματα (διεπαφή χρήστη) εμφανίζονται στον client socket.

**Στο αρχείο Makefile υπάρχει κατάλληλος χώρος που μπορούμε να
παραμετροποιήσουμε τις IP και τις πόρτες των εφαρμογών (by default:
127.0.0.1 και 4001).

***Τέλος με την εντολή "make clean" διαγράφουμε όλα τα εκτελέσιμα
αρχεία.
